\# Telegram Gifts Market (Telegram Mini App на TON)



Полностью статический маркетплейс Telegram Mini App для покупки и продажи \*\*неапгрейженных Telegram-подарков\*\* на блокчейне \*\*TON\*\*.



\- Фронтенд: \*\*HTML + CSS + Vanilla JS (ES6-модули)\*\*  

\- Интеграция: \*\*Telegram WebApp SDK\*\*, \*\*TON Connect UI\*\*  

\- Хостинг: \*\*GitHub Pages\*\*
